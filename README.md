# Bobinette

## Prepare Image

#### Raspberry Pi 3
```
curl -sL https://deb.nodesource.com/setup_8.x | sudo -E bash -
sudo apt-get install nodejs
```

#### Raspberry Pi Zero
```
wget -O - https://raw.githubusercontent.com/sdesalas/node-pi-zero/master/install-node-v.last.sh | bash
sudo apt-get install nodejs
```
